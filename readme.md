## Getting Started

- Install: `npm install -g generator-skylark`
- Run: `yo skylark`
- Run `grunt dist` for building and `grunt server` for preview


## Options

* `--skip-install`

  Skips the automatic execution of `bower` and `npm` after scaffolding has finished.
